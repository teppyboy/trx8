# Trx8

A lightweight Windows utility that applies modular configuration profiles to customize your system.

> [!WARNING]
> Trx8 is still in early stage, expect bugs, breaking changes without notice and other problems. You are encouraged to report any issues occurred though, as it helps the development of this project.

## Config Documentation

Please see [config.example.json](./config.example.json) for an auto-generated example of a working configuration, and additionally [ACTION](./ACTION.md) to see all available actions and the usage.
